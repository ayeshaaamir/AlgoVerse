**AlgoVerse** is a comprehensive, frontend first learning platform designed to master **Data Structures and Algorithms (DSA)** through **interactive visualizations** and **real-world frontend simulations**.

## Core Features & DSA Mapping

### Algorithm Playground
Run and explore algorithms with step-by-step execution.

**Features**
- Custom input support
- Step-by-step execution
- Time & space complexity display

**DSA Used**
- Sorting Algorithms
- Searching Algorithms
- Recursion
- Greedy Algorithms
- Dynamic Programming

---

### Data Structure Visualizer
Interactive visualizations of common data structures.

**Features**
- Insert, delete, update operations
- Animated transitions
- Traversal highlighting

**DSA Used**
- Arrays
- Stack
- Queue & Deque
- Linked List (Singly & Doubly)
- Tree (Binary Tree, BST)
- Graph
- Heap (Min / Max)
- Trie
- HashMap / Set

---

### Sorting & Searching Visualizer
Dedicated visual modules to understand algorithm behavior.

**DSA Used**
- Bubble Sort
- Selection Sort
- Insertion Sort
- Merge Sort
- Quick Sort
- Heap Sort
- Linear Search
- Binary Search

---

### Recursion & Backtracking Explorer
Visualize recursion depth and backtracking paths.

**Features**
- Call stack visualization
- Recursive tree expansion
- Step tracing

**DSA Used**
- Recursion
- Backtracking
- Maze Solver
- N-Queens
- Sudoku Solver

---

### Dynamic Programming Visualizer
Understand DP through visual state transitions.

**Features**
- DP table visualization
- Memoization vs Tabulation comparison

**DSA Used**
- Fibonacci
- Knapsack Problem
- Longest Common Subsequence (LCS)
- Path optimization problems

---

### Real-World Frontend Simulations
Frontend features implemented using DSA internally.

#### Notification System
**DSA Used:** Queue  
Handles notifications in FIFO order.

#### Task Scheduler
**DSA Used:** Heap / Priority Queue  
Schedules tasks based on priority.

#### Autocomplete Search
**DSA Used:** Trie  
Fast prefix-based searching.

#### Undo / Redo System
**DSA Used:** Stack  
Tracks user actions and state history.

#### Route Planner
**DSA Used:** Graph (BFS / DFS / Shortest Path)  
Visualizes routes and node traversal.

---

### Progress Tracker Dashboard
Track completed topics and learning progress.

**Features**
- Topic completion tracking
- Sorting & filtering
- Persistent state

**DSA Used**
- HashMap / Set
- Sorting algorithms

---

### Challenge Mode
Timed challenges to test algorithmic understanding.

**DSA Used**
- Mixed DSA concepts
- Performance & complexity analysis
